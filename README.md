# raymario

raymario is a Super Mario Bros. imitation written in C using [raylib](http://www.raylib.com/).
It was originally created by [VÃ­ctor Fisac](https://github.com/victorfisac).

## Dependencies

* meson (for building)
* raylib
* GLFW3
* OpenGL
* OpenAL-soft

## Build

```
meson builddir
cd builddir
ninja
sudo ninja install
raymario
```
